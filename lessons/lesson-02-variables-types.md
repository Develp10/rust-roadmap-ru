# Урок 2. Переменные, типы данных, выражения

## Теория

### Иммутабельность по умолчанию

Объявление переменной: `let x = 5;`. По умолчанию переменные неизменяемы. Чтобы изменять — нужно `let mut x = 5;`. Это сознательное проектное решение: компилятор помогает находить места, где значение меняется.

### Shadowing (затенение)

```rust
let x = 5;
let x = x + 1;       // новая переменная x, прежняя забыта
let x = "пять";      // даже тип может смениться
```

Shadowing отличается от `mut`: создаётся новый биндинг с новым типом и владением, прежний попадает в drop.

### Константы

```rust
const MAX_USERS: u32 = 100_000;
```

Тип обязателен. Вычисляются на этапе компиляции, доступны как константные выражения. `static` — другая сущность: имеет адрес в памяти и время жизни `'static`.

### Скалярные типы

Целые: `i8 i16 i32 i64 i128 isize` и `u8..u128 usize`. Литералы: `1_000`, `0xff`, `0b1010`, `b'A'`. Размер по умолчанию для целочисленных литералов — `i32`.

Числа с плавающей точкой: `f32`, `f64` (по умолчанию).

`bool` — 1 байт. `char` — 4 байта, любой Unicode scalar value (U+0000..U+D7FF и U+E000..U+10FFFF).

### Составные типы

Кортежи: `let t: (i32, f64, char) = (1, 2.0, 'a');`, доступ `t.0`.

Массивы фиксированного размера: `let a: [i32; 5] = [1,2,3,4,5];`. Размер — часть типа.

### Выражения и блоки

Rust — выражение-ориентированный язык. Блок `{ ... }` — это выражение: возвращает значение последней строки без `;`.

```rust
let y = {
    let n = 3;
    n * n           // обратите внимание — нет ;
};
```

`if` тоже выражение: `let s = if cond { "yes" } else { "no" };`.

## Практика

Реализуйте функцию, считающую статистику массива из 5 чисел:

```rust
fn stats(arr: [i32; 5]) -> (i32, i32, i32) {
    let min = *arr.iter().min().unwrap();
    let max = *arr.iter().max().unwrap();
    let sum: i32 = arr.iter().sum();
    (min, max, sum)
}

fn main() {
    let (mn, mx, s) = stats([3, 1, 4, 1, 5]);
    println!("min={mn}, max={mx}, sum={s}");
}
```

Дополните: добавьте подсчёт среднего как `f64`, верните кортеж из 4 элементов и распечатайте через именованный форматтер `{name}`.

## Тест

**1.** Чем shadowing принципиально отличается от `mut`?

**2.** Сколько байт занимает `char` и почему именно столько?

**3.** Что выведет программа?

```rust
fn main() {
    let x = 5;
    let x = x + 1;
    let x = x * 2;
    println!("{x}");
}
```

**4.** Скомпилируется ли?

```rust
fn main() {
    let mut x = 5;
    let x = "пять";
    println!("{x}");
}
```

**5.** Что не так с этим кодом?

```rust
fn main() {
    let n = 5;
    let y = {
        let n = n + 1;
        n * 2;
    };
    println!("{:?}", y);
}
```

---

### Ответы

1. `mut` позволяет изменять значение по тому же биндингу — тип не меняется. Shadowing создаёт **новую переменную**: меняется тип, владение, и старая уходит в drop. Shadowing разрешён даже без `mut`.

2. 4 байта. `char` хранит любой Unicode scalar value (до `U+10FFFF`), это диапазон не помещается в 1 или 2 байта. Это не байт UTF-8, а кодовая точка.

3. `12`. Каждый `let x = ...` создаёт новый биндинг.

4. Скомпилируется. Shadowing разрешён независимо от `mut` и может менять тип переменной. `mut` от первого `let` просто становится неактуален, потому что новый `let` затеняет переменную.

5. `y` имеет тип `()`: внутри блока стоит `;` после `n * 2`, поэтому блок возвращает unit. Чтобы получить число, нужно убрать точку с запятой у последней строки.
